# Note-Taking App back-end

Secure REST API for note-taking app using **Auth0**, **MongoDB**, **Express.js**.

## Features

- ✅ **CRUD operations** for user-specific notes (`/api/notes`)
- ✅ **Auth0 authentication** (JWT tokens)
- ✅ **Swagger API docs** (`/api-docs`)
- ✅ **User isolation** (each user sees only their notes)
- ✅ **CORS enabled** (works with any front-end)

## Quick Start

```bash
npm install
# Update .env with your Auth0/MongoDB credentials
npm start
```
